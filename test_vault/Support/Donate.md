# Donate

`mdgarden` is free, open-source software released under
[GPL-3.0-or-later](https://github.com/THANSHEER/mdgarden/blob/main/LICENSE).
It costs nothing to use, but it does cost time to maintain — if it's useful
to you, here's how to help.

## Ways to support

- ⭐ **Star the repo** — [github.com/THANSHEER/mdgarden](https://github.com/THANSHEER/mdgarden)
- 🐛 **Report bugs** — see [[Support/Reporting Issues|Reporting Issues]]
- 🤝 **Contribute code or docs** — see [[Support/Contributing/Contribution Guide|Contributing]]
- 💸 **Sponsor** — a sponsorship link will be added here once set up
